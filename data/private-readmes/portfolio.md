# Portfolio — jaqbek.eth

> This portfolio — a Next.js application combining a DOS amber terminal UI, dual AI chat, GitHub integration, and a Python RAG microservice.

GitHub: https://github.com/0xjaqbek/newPortfolio

## About

More than a business card: a personal presentation layer, a repository knowledge catalogue, and an AI/RAG experiment in one. Features a DOS-style amber terminal interface with animated boot sequence, a dual AI chat (DeepSeek direct + RAG assistant), real-time GitHub contribution data, and a Guardian security system with audit logging and IP blocking.

## Key Features

- DOS amber terminal UI with animated boot sequence and Norton Commander-style navigation
- Dual AI chat: DeepSeek direct API and RAG-enhanced semantic search
- RAG assistant backed by Python/FastAPI/ChromaDB with sentence transformer embeddings
- GitHub integration via Octokit: live repos, contribution map, private README notes
- Guardian security system: prompt injection detection, session management, IP blocking, audit logs
- Profile, experience, projects and skills served from JSON data layer
- AppSettings context: boot sequence toggle, AI provider selector
- CSS Modules with full DOS amber palette and scanlines overlay

## Tech Stack

Next.js 14 · React · TypeScript · CSS Modules · Prisma · Octokit · DeepSeek API · Python · FastAPI · ChromaDB · Sentence Transformers · Docker · Heroku · Railway
