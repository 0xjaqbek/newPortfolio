# meldChecker

Link: https://github.com/0xjaqbek/meldChecker

Daty: utworzone 2024-10-31, ostatni push 2024-11-20.

## Tech stack

- JavaScript, React 18, Vite
- Firebase
- Solana web3.js
- wagmi, viem, ethers
- Web3Modal / WalletConnect
- Tailwind CSS, shadcn/ui / Radix UI

## Funkcje / zawartosc

MELD Token Checker: aplikacja webowa do sprawdzania salda tokena Solana w portfelu w celu kontroli dostepu do prywatnej grupy Telegram. README opisuje codzienne kontrole i usuwanie osob, ktore sprzedaly tokeny.
