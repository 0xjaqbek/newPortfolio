# memory-keeper

Link: https://github.com/0xjaqbek/memory-keeper

Widocznosc: prywatne. Daty: utworzone 2026-05-10, ostatni push 2026-05-13.

## Tech stack

- TypeScript
- Vite + React
- TanStack Router + TanStack Query
- Tailwind CSS 4
- Radix UI / shadcn-like component stack
- Supabase
- Cloudflare Vite plugin

## Funkcje / zawartosc

Nazwa i stack wskazuja na aplikacje webowa do przechowywania lub porzadkowania wspomnien/notatek. Brak opisowego README, ale konfiguracja pokazuje SPA/SSR-like frontend z routingiem plikowym TanStack Router i backendem/danymi przez Supabase.
