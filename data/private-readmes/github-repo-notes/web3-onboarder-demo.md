# web3-onboarder-demo

Link: https://github.com/0xjaqbek/web3-onboarder-demo

Daty: fork utworzony 2025-09-05, ostatni push 2025-09-06.

## Tech stack

- Next.js 15 + TypeScript
- React 18
- Tailwind CSS
- Coinbase OnchainKit / MiniKit
- Farcaster Mini App SDK
- wagmi, viem
- Upstash Redis
- Stream Chat / Stream Video

## Funkcje / zawartosc

Fork/demonstracja onboardingu Web3 w formacie Farcaster Mini App. README opisuje MiniKit, OnchainKit, metadane Frame, account association, Redis webhooki i powiadomienia.
