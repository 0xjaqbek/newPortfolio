# pasnik

Link: https://github.com/0xjaqbek/pasnik

Daty: utworzone 2026-02-13, ostatni push 2026-02-13.

## Tech stack

- Next.js 16 + React 19 + TypeScript
- tRPC v11
- Prisma 7 + PostgreSQL
- TanStack React Query
- Tailwind CSS 4
- PWA, web-push
- Zod, JWT, bcrypt
- Vitest

## Funkcje / zawartosc

PWA do zarzadzania zywnoscia: spizarnia z datami waznosci, przepisy, generowanie przepisow przez AI z dostepnych produktow, kalendarz posilkow, lista zakupow i powiadomienia push o terminach, brakach i planach posilkow.
