# plonbli

Link: https://github.com/0xjaqbek/plonbli

Widocznosc: prywatne. Daty: utworzone 2026-03-27, ostatni push 2026-05-13.

## Tech stack

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4
- Drizzle ORM + Neon serverless Postgres
- Supabase, Firebase Admin
- NextAuth beta
- next-intl, next-pwa
- Leaflet / React Leaflet
- React Hook Form + Zod
- QR code, react-pdf
- Vitest / Testing Library

## Funkcje / zawartosc

Glowna aplikacja Plonbli. Manifest sugeruje rozbudowany produkt webowy/PWA z autoryzacja, internacjonalizacja, mapa, baza danych, generowaniem PDF/QR i testami. README jest domyslnym README Next.js, wiec funkcje domenowe nie sa opisane wprost.
