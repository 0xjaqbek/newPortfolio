# apteczka

Link: https://github.com/0xjaqbek/apteczka

Daty: utworzone 2025-09-19, ostatni push 2025-09-30.

## Tech stack

- Vue 3
- Vite
- Pinia
- Vue Router
- Firebase Auth / Firestore / Hosting
- QR code
- vite-plugin-pwa

## Funkcje / zawartosc

PWA do zarzadzania lekami i sprzetem w karetkach. README opisuje autoryzacje, zarzadzanie karetkami, role admin/czlonek, inwentarz, daty waznosci, powiadomienia, historie aktywnosci, kody QR, ciemny motyw i instalacje jako aplikacja mobilna.
