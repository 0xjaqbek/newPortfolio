## Funkcje / zawartosc

System treningowy z panelami zawodnika, trenera i admina. README opisuje kalendarz treningow, dziennik treningowy, feedback, statystyki, zawody/starty, wiadomosci w czasie rzeczywistym, widok zawodnika dla trenera i zarzadzanie treningami.
