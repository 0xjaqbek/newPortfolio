# ragrma

Link: https://github.com/0xjaqbek/ragrma

Widocznosc: prywatne. Daty: utworzone 2026-01-23, ostatni push 2026-01-31.

## Tech stack

- Python
- FastAPI + Uvicorn
- SQLAlchemy + Alembic
- PostgreSQL przez psycopg2
- ChromaDB
- sentence-transformers
- OpenAI SDK
- Jinja2
- JWT / passlib / bcrypt
- pytest

## Funkcje / zawartosc

Backend Pythonowy z elementami RAG/AI: FastAPI, wektorowa baza ChromaDB, embeddingi sentence-transformers i integracja OpenAI. Wyglada na aplikacje z autoryzacja, baza SQL i warstwa wyszukiwania/generowania odpowiedzi.
