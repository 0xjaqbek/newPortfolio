# ekodirect

Link: https://github.com/0xjaqbek/ekodirect

Daty: utworzone 2025-05-15, ostatni push 2025-05-20.

## Tech stack

- TypeScript, React 19, Vite
- Tailwind CSS
- React Router, Zustand, React Hook Form, Zod
- TanStack React Query, Axios
- Stripe
- Leaflet / React Leaflet
- Express, firebase-admin
- QR code

## Funkcje / zawartosc

Platforma marketplace dla ekologicznych produktow/rolnikow. Manifest wskazuje na frontend React, mapy, formularze, QR, platnosci Stripe i element backendowy przez Express/Firebase Admin. README jest glownie szablonowe.
