# ekodirekt

Link: https://github.com/0xjaqbek/ekodirekt

Daty: utworzone 2025-05-13, ostatni push 2025-05-13.

## Tech stack

- Monorepo npm workspaces
- Frontend: React + TypeScript + Vite, Tailwind CSS, Zustand, React Router, React Query
- Backend: Node.js + Express + TypeScript
- MongoDB + Mongoose
- JWT auth
- Multer + Cloudinary
- Swagger/OpenAPI
- Stripe

## Funkcje / zawartosc

Platforma laczaca ekologicznych rolnikow z konsumentami. README opisuje wystawianie produktow, zamowienia lokalnej zywnosci, sledzenie pochodzenia przez QR, funkcje spolecznosciowe i klimatyczne.
