# aptekazrm

Link: https://github.com/0xjaqbek/aptekazrm

Daty: utworzone 2026-04-02, ostatni push 2026-04-03.

## Tech stack

- Next.js 16 + TypeScript
- React 19
- Supabase
- html5-qrcode
- IndexedDB przez `idb`
- Zustand
- react-qr-code
- Tailwind CSS 4

## Funkcje / zawartosc

Aplikacja dla apteczki lub magazynu ZRM. Zaleznosci wskazuja na uwierzytelnianie i dane w Supabase, skanowanie QR, generowanie QR, stan lokalny i prace offline przez IndexedDB.
