## Funkcje / zawartosc

TRIAGE MCI to PWA do segregacji i zarzadzania zdarzeniem masowym. README opisuje algorytm START, kategorie T1-T4, dokumentacje obrazen, tryb dyspozytora, formularze dyslokacji poszkodowanych, tabele szpitali i raport GDM. Aplikacja dziala offline na iOS, Androidzie i Windows oraz obsluguje kilka jezykow.
