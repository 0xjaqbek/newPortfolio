# Dr Kolo — Bike Repair Shop App

> Full-stack web application for a bike repair shop: public site, service pricing, repair orders, bookings, and AI chat.

GitHub: https://github.com/0xjaqbek/drkolo

## About

End-to-end client project covering the public landing page, service pricing driven by `cennik.json`, repair order creation and tracking via unique hash links, appointment booking with admin calendar, a customer questionnaire, and an integrated AI chat widget with admin panel. Backend on Supabase with Vercel Functions as API proxy.

## Key Features

- Public landing page with gallery, service offer and branding
- Pricing page driven by cennik.json
- Repair order creation and status preview via hash link (no login needed for customers)
- Appointment booking with admin calendar view
- Customer questionnaire with admin response view
- In-page AI chat widget and ChatAdmin panel
- Chat API with system prompt, SMS parsing and Supabase integration
- Component, hook and SMS-parser unit tests with Vitest

## Tech Stack

Vite · React · TypeScript · React Router · TanStack Query · Tailwind CSS · shadcn/ui · Supabase · Vercel Functions · Vitest
