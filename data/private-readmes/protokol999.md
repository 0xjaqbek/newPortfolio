# Protokół 999

> AI-powered medical emergency training platform for paramedic students

**Gallery:** 4 screenshots available

## Overview

Protokół 999 is a comprehensive e-learning platform specifically designed for medical and paramedical education. The platform provides realistic medical emergency scenarios with intelligent assessment and progress tracking aligned with ERC 2021 Guidelines.

## Technologies

- Next.js 15
- TypeScript
- React
- AI Integration (OpenAI API)
- Firebase
- Tailwind CSS

## Key Features

### Interactive Learning
- AI-powered scenario engine with real-time feedback
- Realistic medical emergency simulations
- Adaptive difficulty based on student performance
- Multi-choice and open-ended question formats

### Progress Tracking
- Comprehensive analytics dashboard
- Performance metrics and statistics
- Historical progress visualization
- Skill gap identification

### Professional Tools
- Scenario builder for instructors
- Content management system
- Student performance reports
- ERC 2021 Guidelines compliance

### Platform Features
- 24/7 cloud-based accessibility
- Mobile-responsive design
- Secure authentication system
- Real-time data synchronization

## Impact

Serving medical students across Poland with a modern, AI-enhanced approach to emergency medical training. The platform bridges the gap between theoretical knowledge and practical application through intelligent, scenario-based learning.
